01 / Overview

What This Model Does

Binary Classification

Predicts whether an adverse event report involves serious outcomes — hospitalization, death, or life-threatening events.

Dual Model Approach

XGBoost gradient-boosted trees and Logistic Regression with balanced class weights for robust comparison.

Rich Feature Space

Age normalization, gender encoding, TF-IDF symptom vectors (500 terms), and FDA industry categorization across 30+ groups.

2004–2017 Q2 Dataset

90,786 FDA CAERS reports spanning 13 years of adverse event data from the Center for Adverse Event Reporting System.
